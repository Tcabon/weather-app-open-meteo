# Weather App

Check the current weather on any city on the planet. Switch between metric and imperial units.

## Features

1. User's ability to display location weather

2. Current local time and date

3. Temperatures and humidity

4. Wind speed and direction

5. Sunrise and sunset times

6. Metric vs Imperial system

7. Error handling and loading info

## Installation

1. `git clone git@github.com:Tcabon/weather-app-open-meteo.git`

2. `cd weather-app-open-meteo`

3. `npm install`

4. `Open city.json in the config folder and enter location informations (you can find the informations of the city you are looking for at the start of https://open-meteo.com/en/docs)`

5. `npm run dev`
